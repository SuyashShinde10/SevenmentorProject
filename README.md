SaaS Employee Management System

A full-stack **MERN (MongoDB, Express, React, Node.js)** application designed as a **SaaS (Software as a Service)** platform. This system allows multiple companies (HR) to register, manage their employees, assign grouped tasks, and track progress in real-time.
